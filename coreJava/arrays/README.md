## 1) Maximum in an array <br/>
   You are give an array of size N. N is an integer number you need to find maximum value in given array. 
### Input
        Enter an array 
### Output
        return a number

### Sample input 1 
         [ 3 , 7 , 1 , 6 ,  4 ]
### Sample Output 1
         7
